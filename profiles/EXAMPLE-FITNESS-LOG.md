## 训练记录格式

每次训练完成后，在此追加一条记录：

```
## [YYYY-MM-DDTHH:MM:SS+08:00] <训练部位 / 动作名称>
- **exercises**: [动作列表]
- **rpe**: <自觉疲劳系数 1-10>
- **duration_minutes**: <训练时长>
- **notes**: 
    - <训练感受、PR 突破、体感笔记>
    - <RPG 属性更新>
```

### 示例

```
## [2026-05-01T18:30:00+08:00] 推类日（胸+三头）
- **exercises**: [杠铃卧推, 哑铃上斜, 双杠臂屈伸, 仰卧臂屈伸, 侧平举]
- **rpe**: 8
- **duration_minutes**: 60
- **notes**:
    - 卧推正式组 45kg x 8次 x 3组，RPE 8
    - 上斜哑铃 14kg x 12次 x 4组
    - 体感良好，胸肌发力感明显
    - [RPG] STR +0.5, EXP +25
```
