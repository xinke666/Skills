## 详细介绍

### 周六野 Zoey

方向：塑形、减脂、全身训练。她的训练以"无跳跃、不伤膝、居家可做"著称。

- **风格**：温柔鼓励，像朋友陪你练。强调"坚持比强度重要"
- **适合你**：想要温和但有效的训练，不喜欢被吼，喜欢低冲击动作
- **不太适合**：追求极限力量增长、或喜欢高强度CrossFit风格
- **平台**：B站、小红书、YouTube

### 帕梅拉 Pamela Reif

方向：HIIT、全身塑形、核心。10-30分钟视频信息密度极高，音乐驱动。

- **风格**：高效无废话。动作标准高、节奏快
- **适合你**：有一定运动基础，时间紧张但想高效训练
- **不太适合**：完全零基础、有关节问题、或不喜欢高强度间歇训练
- **平台**：B站（官方中文频道）、小红书、Keep

### 林芊妤 Coffee Lam

方向：瑜伽、瑜伽塑形、拉伸。介于传统瑜伽和现代塑形之间——不是纯冥想，也不是纯流汗。

- **风格**：舒缓专注，重视呼吸与控制，追求"线条变长变美"
- **适合你**：喜欢瑜伽，想要柔韧性和力量的结合，想改善体态
- **不太适合**：只想要大汗淋漓的燃脂训练
- **平台**：YouTube（主阵地）、B站

### 欧阳春晓

方向：瘦腿、体态矫正、局部塑形。内容非常细分——假胯宽、XO型腿、骨盆前倾、圆肩驼背等。

- **风格**：细致拆解每个动作的"为什么"，强调正确发力感
- **适合你**：有特定体态问题想要针对性改善
- **不太适合**：追求全身燃脂或力量增长（她的内容偏局部矫正）
- **平台**：小红书（主阵地）、B站

### 韩小四

方向：温和减脂、新手入门。所有推荐教练中强度最低的一个。

- **风格**：超友好，全程站立为主（不用趴地上），每做完一组就鼓励你
- **适合你**：完全零基础、BMI偏高、有关节顾虑、需要"超温和"起步
- **不太适合**：已有运动基础需要进阶挑战
- **平台**：B站

### 海洋饼干

方向：减脂、全身塑形。训练和饮食建议并重，不只是训练博主。

- **风格**：直接接地气，像私教。不绕弯子
- **适合你**：想要"直说"风格，既练也讲吃，中等强度
- **不太适合**：喜欢温柔鼓励风（周六野更适合）
- **平台**：B站、小红书

## 怎么选？

### 按训练目标

| 目标 | 推荐 |
|:---|:---|
| 减脂 | 周六野、海洋饼干 |
| 塑形 | 周六野、帕梅拉 |
| 高强度 | 帕梅拉 |
| 瑜伽柔韧 | 林芊妤 |
| 体态矫正 | 欧阳春晓 |
| 新手入门 | 韩小四 |
| 增肌力量 | 凯圣王×谭指导 |

### 按当前基础

| 基础 | 推荐顺序 |
|:---|:---|
| 零基础 | 韩小四 → 周六野 → 帕梅拉 |
| 有一定基础 | 周六野 → 帕梅拉 / 海洋饼干 |
| 进阶 | 帕梅拉 + 凯圣王×谭指导（力量日） |

### 按风格偏好

- **喜欢鼓励陪伴** → 周六野、韩小四
- **喜欢高效直接** → 帕梅拉
- **喜欢搞懂原理** → 欧阳春晓
- **喜欢瑜伽氛围** → 林芊妤
- **喜欢私教感** → 海洋饼干

## 混搭建议

不必只跟一个教练。很多人的最佳方案是混搭：

- **力量日**用凯圣王的推拉腿计划 — 想要力量和肌肉线条
- **有氧日**用帕梅拉的HIIT — 想要心肺和燃脂
- **恢复日**用林芊妤的瑜伽拉伸 — 想要柔韧性和放松
- **体态矫正**穿插欧阳春晓 — 想要改善特定问题

初始化时告诉 AI 你的混搭偏好，例如："我想力量日跟凯圣王，有氧跟帕梅拉，休息日跟林芊妤拉伸。"
