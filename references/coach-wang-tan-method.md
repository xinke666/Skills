> 适用人群：新手至中级训练者
> 核心理念：力量 + 形体 + 功能性三位一体，循证科学，拒绝蛮干

## 一、训练分化结构（三分化 / Push-Pull-Legs 变体）

### Day 1：推类（胸 + 三角肌中束 + 三头）
| 序号 | 动作 | 定位 | 初始参数 |
|------|------|------|----------|
| 1 | 杠铃卧推（主项） | 上肢推力主线 / 神经开发 | 热身15次 + 正式组 12-10-8 次，RPE 8（留 2 次余量） |
| 2 | 哑铃上斜卧推 | 上胸堆肉 | 4x12，RPE 8，后期末 1-2 组可接近力竭 |

| 5 | Y字外展侧平举 | 三角肌中束 | 3x(10+10) 力竭制（做10次→歇5秒→再10次力竭） |

| 序号 | 动作 | 定位 | 初始参数 |

| 3 | 单手器械划船 | 水平拉 / 背阔肌 | 3x10 不力竭 + 末组 10 次力竭+5次，逐周递增力竭组 |

## 二、核心训练哲学（必须遵守）

### 1. 力竭管理（最重要的一点）
- **大重量复合动作 → 不力竭**：留 RPE 8（约 2 次余量），保护神经系统，保证训练频率。
- **小重量孤立动作 → 可力竭**：如侧平举、弯举等单关节动作，神经消耗小，力竭可募集更多白肌纤维。
- **判断标准**：该动作能否借力？重量轻不轻？如果无法借力且重量不大 → 力竭安全。

### 2. 线性递进逻辑（渐进超负荷的具体方法）
主项（如杠铃卧推）的递进路径：
```
第1周：热身15次 → 12次 → 10次 → 8次（RPE 8）
第2周：热身15次 → 8次 → 10次 → 8次（将12次组替换为8次）
第3周：热身15次 → 8次 → 8次 → 8次（全部正式组变8次）
第4周：热身 → 4x8（强度+容量均高）
接下来：降容量、加强度 → 2x6 RPE 8（回归低量高强度）
然后：2x6 → 3x6 → 4x6 → 再降至 2x5 → 循环...
```
- **关键**：强度提升的同时容量先降，再慢慢加回来，避免神经爆掉。

### 3. 辅助动作递进逻辑
```
第1周：4x15 RPE 8（不力竭）
第2周：3x15 + 1x12（逐渐加入12次组）
第3周：2x15 + 2x12
...
最终：4x12，末 1-2 组接近力竭 → 微加重量 → 重新开始
```

### 4. 自由重量的意义
- 不只是增肌工具，更是**身体能力的评估手段**。
- 杠铃卧推可以筛查：肩胛后缩能力、胸椎活动度、肱骨位置异常、外展代偿等。
- 遇到问题 → 发现问题 → 解决问题 → 水平提升。

### 5. 休息与频率
- 三分化适合新手（频率高、强度适中）。
- **绝不硬跟计划**：疲劳时，哪怕只练了2天也要休息。
- 监测手段：晨脉、HRV（BDG）等。
- 练4天歇1天**不推荐**，练3天歇1天可以但因人而异。
- "计划之所以叫计划，就是它一定要符合你自己。"

### 6. 动作替代规则
- 1个月后可根据疲劳做微调：
  - 肩屈位臂屈伸 → 肩伸位下压
  - 二头内侧 → 二头外侧
  - 神经积累高时 → 加肩部容量来释放推类的神经强度

## 三、教练风格特征

### 王凯圣（凯哥）
- **风格**：循证科学派，强调神经适应和疲劳管理
- **偏好**：自由重量评估身体能力、渐进式开发
- **口头禅**："这东西不考验毅力，它是循证科学"

### 谭指导（谭教练）
- **风格**：技术派，强调动作感觉和肌纤维走向
- **偏好**：钢线弯举（肩屈位）、海豹划船、动作控制
- **口头禅**："一定要有感觉，不能抡、不能甩"

## 四、2026 精英版专项补完 (Lv.3+ 进阶建议)

### 1. 谭氏动态热身协议 (Pre-Workout)
- **目的**：激活神经招募，而非消耗体力。
- **推日**：弹力带肩外旋 2x20，空杆卧推（底部停顿 2s）2x15。
- **拉日**：猫式伸展 1min，悬垂肩胛收缩 2x12。
- **腿日**：自重深蹲（底部侧移）2x15，死虫式 2x10。

### 2. [左手之影] 力量平衡协议
- **触发条件**：单侧动作左右差异 > 2 次或 > 10% 重量。
- **执行规则**：
    1. **弱侧先行**：永远从非优势手（通常是左手）开始。
    2. **强侧迁就**：强侧的重量和次数必须严格匹配弱侧，禁止超前。
    3. **专项加餐**：在拉日结束前，额外增加 1-2 组弱侧的孤立收缩练习。

### 3. 高级力竭技巧 (Elite Techniques)
- **递减组 (Drop Sets)**：仅限最后 1 个孤立动作。力竭后降重 30%，立刻继续直到再次力竭。
- **休息暂停 (Rest-Pause)**：在主项末组，力竭后休息 10s，强行再推 1-2 个。
- **离心控制**：所有动作放回过程保持 2-3 秒，这是“堆肉”的关键。

### 4. 神经系统与恢复监测
- **警报信号**：连续两天晨脉升高 > 5bpm，或主项重量下降 > 10%。
- **应对**：强制执行 Active Recovery（如 04-26 的技巧复习课），禁止强冲重量。
